Plugin WordPress pour afficher et gérer des réservations.

== Description ==
Ce plugin offre la possibilité d'implémenter un calendrier de réservation sur un site WordPress via un shortcode. Il sera nécessaire de l'implémenter sur une page dont le slug est "calendar". Il faudra aussi une page "remerciement" pour la redirection après la réservation.

== Installation ==
Installation classique de l'extension via le téléversement ou via le store WordPress Plugins. 
Créer une page dont le slug est "calendar" et une page dont le slug est "remerciement" pour la redirection après la réservation.
Installer le shortcode [calendar] sur la page "calendar".
